# 1. What is NumPy?

**NumPy = Numerical Python**

Core object:
ndarray (n-dimensional array)

*It is:*
Homogeneous (same data type)

Fast *(written in C under the hood)*
Memory efficient
Designed for mathematical computation
Unlike Python lists:
Lists store references
NumPy arrays store raw contiguous memory
That’s why NumPy is much faster.
