# 📊 Variance & Standard Deviation

This module implements:

* Population Variance
* Sample Variance
* Standard Deviation

---

## 1️⃣ Mean (Average)

μ=1/N​∑xi

Where:

* ( xi ) = each data value
* ( N ) = total number of values

---

## 2️⃣ Population Variance

Population variance measures how far data points spread from the population mean.

σ^2=1/N​∑(xi​−μ)^2

---

## 3️⃣ Sample Variance

Sample variance is used when data is a subset of a larger population.

s^2=1/(N-1)​∑(xi​−xˉ)

---
